Module weathermetrics

weathermetrics module for Lua 5.1. weathermetrics includes functions to calculate dew point temperature, relative humidity, and heat index and to convert between Celsius and Fahrenheit. All functions are based on equations and algorithms used by the United States National Weather Service. License: MIT

celsius_to_fahrenheit (celsius, digits)
Convert from Celsius to Fahrenheit. Create a temperature in Fahrenheit from a temperature in Celsius. Equations are from the source code for the US National Weather Service's online heat index calculator.

Parameters

  • celsius: a temperature value in Celsius
  • digits: number of digits to round converted value (defaults to 2)

Usage:

celsius_to_fahrenheit(celsius, digits)

Return value:

a temperature in Fahrenheit from a temperature in Celsius

  • fahrenheit_to_celsius
dewpoint_to_humidity (dp, t, temperature_metric)
Create a relative humidity value from air temperature and dew point temperature values. Dew point temperature and temperature must be in the same metric (i.e., either both in Celsius or both in Fahrenheit). If necessary, use fahrenheit_to_celsius or celsius_to_fahrenheit to convert before using this function. Equations are from the source code for the US National Weather Service's online heat index calculator.

Parameters

  • dp: dew point temperature
  • t: air temperature
  • temperature_metric: temperature metric for both air temperature and dew point temperature, possible values: FAHRENHEIT or CELSIUS (defaults to FAHRENHEIT)

Usage:

dewpoint_to_humidity(dp, t, temperature_metric)

Return values:

  1. relative humidity (in %)
  2. if the first return value is nil, the second return value contains a message explaining why

See also:

  • fahrenheit_to_celsius
  • celsius_to_fahrenheit
  • humidity_to_dewpoint

heat_index (t, dp, rh, temperature_metric, output_metric, digits)
Calculate heat index. Create a heat index value from air temperature and either relative humidity or dew point temperature values. Include air temperature t and either dew point temperature dp or relative humdity rh . You cannot specify both dew point temperature and relative humidity- this will return an error. Heat index is calculated as nil when impossible values of dew point temperature or humidity are inputted (e.g., humidity above 100% or below 0%, dew point temperature above air temperature).

Parameters

  • t: air temperature
  • dp: dew point temperature
  • rh: relative humidity (in %)
  • temperature_metric: temperature metric for air temperature and, if you're using it, dew point temperature; value can be either FAHRENHEIT or CELSIUS
  • output_metric: metric heat index should be calculated in, either FAHRENHEIT or CELSIUS
  • digits: the number of digits to round the heat index to

Usage:

heat.index(t, dp, rh, temperature_metric, output_metric, round)

Return values:

  1. A heat index value in the metric specified by output_metric (if output_metric is not specified, heat index will be returned in the same metric in which air temperature was input, specified by temperature_metric)
  2. if the heat index value is nil, the second return value will be a message explaining why.
humidity_to_dewpoint (t, rh, temperature_metric)
Calculate dew point temperature. Create a dew point temperature from air temperature and relative humidity. Dew point temperature will be calculated in the same metric as the temperature (as specified by the temperature_metric option). If you'd like dew point temperature in a different metric, use the function celsius_to_fahrenheit or fahrenheit_to_celsius on the output from this function.

Parameters

  • t: air temperature
  • rh: vector of relative humidity (in %)
  • temperature_metric: temperature metric for air temperature, either FAHRENHEIT or CELSIUS (defaults to FAHRENHEIT)

Usage:

humidity_to_dewpoint(t, rh, temperature_metric)

Return values:

  1. the dew point temperatre
  2. if the first return value is nil, the second return value contains a message explaining why.

See also:

  • celsius_to_fahrenheit
